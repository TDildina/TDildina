# Тест-кейс: Авторизация пользователя с проблемными изображениями
## ID
TC_LOGIN_02

## Краткое описание
Проверка входа в систему пользователя с проблемными изображениями товаров.

## Предусловия
- Доступ к интернету.
- Сайт `https://www.saucedemo.com/` доступен.

## Шаги
1. Открыть браузер и перейти по адресу: `https://www.saucedemo.com/`.
2. В поле **Username** ввести: `problem_user`.
3. В поле **Password** ввести: `secret_sauce`.
4. Нажать кнопку **Login**.

## Ожидаемый результат
- Пользователь успешно авторизован.
- Происходит переход на страницу `/inventory.html`.
- На странице отображается заголовок "Products".
- Отображается список товаров как одинаковые плейсхолдеры (проблема пользователя)

## Постусловия
- Выполнить выход из системы (при необходимости) через меню (Logout), чтобы не оставлять активную сессию.

## Приоритет
Средний

## Автоматизирован
Нет

## Примечания
- Пользователь problem_user-специальный тестовый аккаунт, у которого все изображения товаров отображаются одинаково. Тест проверяет корректность авторизации и визуальные особенности интерфейса для данного пользователя.
