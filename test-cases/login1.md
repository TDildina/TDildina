# Тест-кейс: Успешная авторизация пользователя с валидными данными

## ID
TC_LOGIN_01

## Краткое описание
Проверка возможности входа в систему с корректными учётными данными стандартного пользователя.

## Предусловия
- Доступ к интернету.
- Сайт `https://www.saucedemo.com/` доступен.

## Шаги
1. Открыть браузер и перейти по адресу: `https://www.saucedemo.com/`.
2. В поле **Username** ввести: `standard_user`.
3. В поле **Password** ввести: `secret_sauce`.
4. Нажать кнопку **Login**.

## Ожидаемый результат
- Пользователь успешно авторизован.
- Происходит переход на страницу `/inventory.html`.
- На странице отображается заголовок "Products".
- Отображается список товаров (например, "Sauce Labs Bolt T-Shirt").

## Постусловия
- Выполнить выход из системы (при необходимости) через меню (Logout), чтобы не оставлять активную сессию.
