# BR_UI_02: Не выбирается кнопка "No" при наведении курсора

| Поле | Значение |
|------|----------|
| **ID** | BR_UI_02 |
| **Серьёзность** | Medium |
| **Приоритет** | High|
| **Модуль** | Radio-button |
| **Окружение** |Windows 10, 192.168.0.8; Chrome, ver 144.0.7559.133; laptop   |
| **URL** | `https://demoqa.com/radio-button`|

## Preconditions

1. Открыта страница radio-button.
2. Попытаться выбрать кнопку "No".

## Steps to Reproduce

1. Открыта страница radio-button.
2. Попытаться выбрать кнопку "No".

## Actual Result

- Радио-кнопка "No" неактивна, невозможно выбрать этот вариант. Визуально кнопка выглядит неактивной (темной).
   
- При наведении курсора на неактивную кнопку "No" отображается системный курсор not-allowed (перечеркнутый круг), что указывает на невозможность взаимодействия с элементом.

## Expected Result

- Все три варианта (Yes, Impressive, No) должны быть активны и доступны для выбора.

## Recommendations
Для улучшения пользовательского опыта необходимо:
1. Если кнопка должна быть недоступна, то нужно скрыть её из интерфейса.
2. Если кнопка должна отображаться, то нужно активировать её или добавить подсказку о причине блокировки.
